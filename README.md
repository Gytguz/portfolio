# Portfolio

## About This Project

Simple eCommerce site.

## Stack Being Used

- **Laravel 10.x** - backend 
- **Inertia (Vue3)** - frontend
- **Tailwind** - css
- **Mailtrap** - mailing
- **Laravel Cashier** - php side of payments
- **Stripe** - frontend side of paymets
- **Laravel Invoices** - invoices
- **Laravel Shopping Cart** - shopping cart
